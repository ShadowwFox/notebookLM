# notebookLM
O projeto consistiu na construção de um caderno temático no NotebookLM voltado ao estudo introdutório de educação financeira pessoal. Foram selecionadas e incorporadas quatro fontes abertas em texto institucional que abordam conceitos centrais como orçamento, controle de gastos, reserva de emergência e planejamento de metas financeiras.
